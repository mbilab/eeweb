eeweb
=====

# setup
* copy `config.sample.json` to `config.json`
* edit `config.json` to fit your environment
* put your key & crt of ssl into ssl file 
* yarn

# build
* yarn build

# update data
* ./ser.js get

# keep data refresh
* ./ser.js

# develop
* yarn start

# chatbot
* chatbot.php is now on apache in ./dist
